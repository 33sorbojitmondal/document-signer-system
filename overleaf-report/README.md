# Overleaf LaTeX Project Report

## Digital Signature System — Project Report

This folder contains a complete LaTeX project report ready for upload to **Overleaf**.

## How to Use on Overleaf

1. Go to [https://www.overleaf.com](https://www.overleaf.com)
2. Click **New Project** → **Upload Project**
3. Upload the zip file: `Document-Signer-Report-Overleaf.zip`
4. Set the main document to `main.tex` (should be automatic)
5. Click **Recompile** to generate the PDF

## Compiler Settings

- **Compiler:** pdfLaTeX
- **TeX Live version:** 2023 or later (default on Overleaf)

All packages used are standard and available on Overleaf by default.
